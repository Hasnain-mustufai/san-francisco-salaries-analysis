# 📊 Analyse des Salaires Publics de San Francisco

## Pipeline Data Analyst : Python → MySQL → SQL → Power BI

# 🧩 Présentation du projet
Ce projet analyse les salaires publics de la ville de San Francisco (USA) à partir d’un dataset récuperer sur un site.
L’objectif est de construire un pipeline complet de Data Analyst :


* Exploration & Nettoyage (Pandas)
* Intégration & Modélisation SQL (MySQL)
* Analyses avancées (SQL + Window Functions)
* Visualisation professionnelle (Power BI)
* 
Le résultat final est un dashboard interactif permettant d’explorer les postes, les salaires, les avantages et les tendances annuelles.


# 🏗️ Pipeline du projet
1. Import & Exploration (Python / Pandas)
2. Nettoyage & Harmonisation (Data Wrangling)
3. Intégration dans MySQL (table : salaires_sf)
4. Analyses SQL avancées (vues, agrégations, window functions)
5. Dashboard Power BI (KPI + visualisations)

Le résultat final est un dashboard interactif permettant d’explorer les postes, les salaires, les avantages et les tendances annuelles.
## Dashboard Power BI 

Le fichier Power BI  :
- les postes les plus fréquents
- les salaires les plus élevés
- l’évolution annuelle des salaires
- les avantages par poste
