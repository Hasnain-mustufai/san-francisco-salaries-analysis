# san-francisco-salaries-analysis

Projet complet d’analyse des salaires publics de San Francisco.

Le repo contient :

- Nettoyage et préparation des données (Pandas)
- Analyse exploratoire (EDA)
- Analyses statistiques (SQL + Pandas)
- Création de vues SQL optimisées pour Power BI
